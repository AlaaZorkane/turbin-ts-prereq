# Turbin3 Typecript Prereq

### Setup

- Make sure you have `node` & `pnpm` installed
- You can also use `yarn`

### Keygen

```bash
pnpm keygen
```

To export the file into a `dev-wallet.json`:

```bash
pnpm keygen:export
```

### Airdrop

```bash
pnpm airdrop
```

### Transfer

```bash
pnpm transfer
```

### Enroll

You need to have a `Turbin3-wallet.json` file in the root of the project.

```bash
pnpm enroll
```
